Getting Started with Your Managed VPS

Our managed VPS services provide the power of a dedicated server with the ease of use of a shared hosting account. Every VPS is custom-built by our engineers.

  • Root Access: Full administrative control via SSH.
  • Provisioning: High-performance NVMe storage for rapid I/O.
  • Backups: Automated daily snapshots included by default.

SSH Access and Security Keys

Connecting to your server via SSH allows you to manage files and services via the command line.

We strictly recommend using SSH key pairs instead of passwords. Generate a key pair on your local machine then add your public key to ~/.ssh/authorized_keys on your server.

Deploying a Node.js Application

We provide PM2 (Process Manager 2) on all our managed servers for zero-downtime application management.

pm2 start app.js --name "my-app"
pm2 save
pm2 startup

Configuring Nginx Reverse Proxy

Nginx acts as a high-performance entry point for your applications, handling SSL termination and request routing.

Our engineers handle complex Nginx configurations for you, but you can also manage your own vhost files in /etc/nginx/sites-available/.
